Methodology & Caveats

  • Both models were trained on real BSB station data through 30 June 2026. All predictions from 1 July 2026 onward are genuine forecasts, not backtests against real data.
  • The baseline forecast assumes 0 (no known festival/kumbh/exam/pilgrimage) for every event flag across this entire window, since the events calendar built for this project has no coverage past 30 June 2026.
  • Event overlay: selecting a non-zero regressor value above adds a flat, per-2-hour-slot adjustment on top of the baseline forecast. Adjustment sizes come from a separate Prophet regression fitted on the full 2-hourly history with all four event flags included as regressors simultaneously, so each value is an estimate of that event's marginal effect holding the others fixed. This is an additive overlay, not a full re-run of LightGBM/SARIMAX with the new flags baked into their lag/rolling features - treat it as a directional "what-if," not a precise re-forecast.
  • LightGBM forecasts recursively day-by-day using its own prior predictions as history once real data runs out, and remains numerically well-behaved across the full 6-month window (1 Jul - 31 Dec 2026).
  • SARIMAX raw output is fully validated only through 21 July 2026: testing showed its unconstrained point forecasts drift sharply upward beyond roughly 3 weeks out (a known behavior of near unit-root seasonal AR terms combined with a short, compute-constrained training window), reaching several times a realistic level by December. To still provide SARIMAX numbers through 31 December as requested, values beyond 21 July are a stabilized blend: the raw SARIMAX forecast is progressively pulled back toward LightGBM's stable seasonal pattern the further out it goes, so it no longer diverges, but it is no longer the pure unconstrained statistical output either. The results panel flags which of your selected slots fall in this stabilized region.
  • Both models' predictions are always computed together for whatever date/slot range you select, across the full 1 July – 31 December 2026 window. Use the Overlay Chart / Side-by-Side Charts toggle to switch how the comparison is drawn.
